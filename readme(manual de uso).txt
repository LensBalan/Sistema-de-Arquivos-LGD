1º Compilar o código

  - No terminal Linux e no power shel Windows digite: g++ -o main main.cpp

2º Executar o código

  - No terminal Linux e no power shel Windows digite: ./main
    
3º Criando um novo ou abrindo um disco existente
  - Selecione se quer criar ou abrir uma partição através do menu disposto no terminal;

4º Utilizar o Sistema de Arquivos em si
  - Seguindo o menu printado no terminal, escolha entre as 6 opções oq quiser fazer:

  - 1 para Para criar um novo disco;
  - 2 Para adicionar um novo arquivo no disco;
  - 3 Para copiar um arquivo do disco para fora;
  - 4 Para excluir um arquivo do disco;
  - 5 Para listar todos arquivos do disco;
  - 6 Para sair do sistema;

  - OBS: Caso for a primeira execução, pode-se acontecer situações iniciais, como não ter arquivos no disco ainda,
  dessa forma, algumas funcionalidades não terão utilidade nesse primeiro caso, como, por exemplo, excluir
  um arquivo do disco.

5º Observações extras

  - O tamanho mínimo para o disco ser criado é de 34 blocos;
  - O tamanho máximo de criação para o disco é de 8388607 blocos; 
  - Para a cópia de arquivos de fora para dentro do disco, é necessário que os arquivos estejam na mesma pasta do código;
